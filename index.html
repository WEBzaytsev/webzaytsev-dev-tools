        О проекте Dev Tools
        
					
					
        
						
        
							Dev Tools - это набор полезных инструментов для разработчиков, которые помогают автоматизировать рутинные задачи и упростить процесс разработки.
						
        

						
        Доступные инструменты:
        
						
						
        
							
        Генератор портов
        
							
        Автоматически генерирует свободные порты для ваших приложений. Поддерживает:
        
							
          
								
        • Генерацию одиночных портов
          • 
								
        • Генерацию диапазонов портов
          • 
								
        • Запоминание использованных портов
          • 
							
        
						
        

						
        
							
        Docker Run в Docker Compose
        
							
        Конвертирует команды Docker Run в формат Docker Compose. Возможности:
        
							
          
								
        • Поддержка всех основных параметров Docker
          • 
								
        • Уведомления о неподдерживаемых опциях
          • 
								
        • Экспорт в docker-compose.yml
          • 
							
        
						
        

						
        
							
        Генератор токенов
        
							
        Создает криптографически стойкие токены. Особенности:
        
							
          
								
        • Настраиваемая длина токена
          • 
								
        • Выбор наборов символов
          • 
								
        • Мгновенная генерация
          • 
							
        
						
        

						
        
							
        Генератор Lorem Ipsum
        
							
        Продвинутый генератор текста-заполнителя с широкими возможностями настройки:
        
							
          
								
        • Поддержка русского и английского языков
          • 
								
        • Расширенный словарь латинских и русских слов
          • 
								
        • Регулируемая "жёсткость" текста для технического юмора
          • 
								
        • Настройка количества параграфов и предложений
          • 
								
        • Классические начальные фразы для обоих языков
          • 
								
        • Мгновенное копирование результата
          •
